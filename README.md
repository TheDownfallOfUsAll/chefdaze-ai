# 🍳 Cook de Staku – AI-Powered Smart Recipe Assistant

**Cook de Staku** is a modern, interactive cooking assistant built with **Streamlit**. It helps home cooks generate recipes, plan meals, substitute ingredients, and receive smart cooking guidance.

With **12 unique AI personas**, each with a colored icon, cooking becomes more engaging and interactive. Whether you're a beginner or an experienced cook, **Cook de Staku** simplifies meal planning, recipe creation, and kitchen organization through a clean and user-friendly interface.

---

# 🌟 Features

### 🍳 Recipe Generator

Enter your ingredients and receive a **step-by-step recipe** with cooking instructions, tips, and serving information.

### 🔄 Ingredient Substitute

Find **alternative ingredients** for common cooking items such as eggs, milk, or butter.

### 📅 Meal Planner

Generate a **weekly meal plan** with balanced and varied dishes.

### 🥗 Nutrition Calculator

View estimated **calories, protein, carbohydrates, and fat per serving**.

### 🛒 Auto Grocery List Generator

Automatically create a **shopping list** based on the ingredients used in your recipe.

### 📸 Recipe Images

Displays **visual previews of meals** to make recipes easier to follow and more appealing.

# 🚀 Installation

Follow these steps to run **Cook de Staku locally**.

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/cook-de-staku.git
cd cook-de-staku
```

---

## 2️⃣ Create a Virtual Environment (Recommended)

```bash
python -m venv venv
```

Activate the environment:

### Windows

```bash
venv\Scripts\activate
```

### macOS / Linux

```bash
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

**requirements.txt**

```
streamlit
```

---

## 4️⃣ Run the Application

```bash
streamlit run app.py
```

---

## 5️⃣ Open the Application

After running the command, Streamlit will provide a local link such as:

```
http://localhost:8501
```

Open this link in your browser to start using **Cook de Staku**.

# 🔄 Chatbot Workflow

Below is a simplified outline of how the **Cook de Staku chatbot** processes user requests.

```mermaid
flowchart TD

A[Start] --> B[User opens Cook de Staku]
B --> C[User selects mode]
C --> D[User enters ingredients or question]
D --> E[System receives input]
E --> F[Keyword Detection]

F --> G{Contains "substitute"?}
G -->|Yes| H[Call substitute_ingredient()]
H --> I[Generate substitute list]

G -->|No| J{Contains "meal plan"?}
J -->|Yes| K[Call meal_plan()]
K --> L[Generate weekly meal plan]

J -->|No| M[Call generate_recipe()]
M --> N[Generate recipe + nutrition + grocery list]

I --> O[Assign random AI persona]
L --> O
N --> O

O --> P[Display response in chat]
P --> Q[Store chat history]
Q --> R[Wait for next user input]
```
